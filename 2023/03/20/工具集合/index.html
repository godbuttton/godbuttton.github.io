Intellij Idea 常用插件

插件名 用处 备注
Translation 随时翻译
Alibaba Java Code Guidelines 阿里巴巴 Java 代码规范，帮助你代码更符合规范，建议安装后将语言替换为中文，提示更加人性。 在 idea 中双击 shit 输入 阿里，切换中英文
CodeGlance3 生成代码地图
MyBatisX IDEA 的快速开发插件，方便在使用mybatis以及mybatis-plus开始时简化繁琐的重复操作
Search In Repository 把 Maven 的中央仓库的查找功能集成到 idea 中
GSONFormat 可以利用 GsonFormat插件，解析Json格式数据快速创建符合 Gson要求的 JavaBean
Maven Helper 开发过程中，我们难免会遇到各种jar包冲突，Maven Helper可以很方便的找到和排除冲突的依赖。
Grep Console 控制台日志颜色
chinese simple 语言包
